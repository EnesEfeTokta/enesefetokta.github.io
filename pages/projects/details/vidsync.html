Project Overview

VidSync is a cutting-edge video meeting platform that combines robust real-time collaboration with intelligent AI assistance. Built with modern web technologies, it delivers seamless video conferencing, instant messaging, and automated meeting insights.

The platform uses Mistral 7B, a powerful local LLM, to automatically generate meeting summaries and extract action items—ensuring no important detail is lost.

Key Features

  • AI Meeting Assistant

    Automated conversation capture and summarization using Mistral 7B LLM.

  • HD Video Conferencing

    Crystal-clear P2P video streaming powered by WebRTC technology.

  • Enterprise Security

    End-to-end encryption with JWT-based authentication.

  • Real-time Messaging

    Instant sync using SignalR websocket connections.

  • Redis Caching

    High-performance session and state management.

  • Containerized

    Full Docker support for seamless deployment.

Technical Highlight: Async AI Integration

One of the biggest challenges was integrating Mistral 7B without degrading video performance. Running an LLM requires significant resources, which could block the main application thread.

Solution: Microservices Architecture

I implemented an asynchronous queue-based system where AI processing happens in a separate Python service:

  • Meeting audio is transcribed in 30-second windows
  • Transcripts are pushed to a message queue
  • The Python AI service consumes and processes with Ollama
  • Results are pushed back via SignalR in real-time

This ensures the main thread stays free for video/audio streams, maintaining a smooth user experience.
