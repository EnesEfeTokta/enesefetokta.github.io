Fishing Game

A relaxing yet challenging spear fishing game for Android, Web, and Windows

Unity 3D C# WebGL Physics System Particle FX
Play on itch.io View Source

Play directly in your browser via WebGL or download for Windows/Android

Game Mechanics

Test your precision and reflexes by hunting fish with well-aimed spear throws. The game features a dynamic spawning system where difficulty scales across 20 unique levels, introducing faster fish species and tighter time constraints.

Built with Unity's physics engine for realistic projectile motion and particle effects for visual feedback on successful catches.

Game Features

  • Precision Physics

    Realistic spear trajectory using Unity's physics system.

  • Dynamic Spawning

    8 unique fish types with varying speeds and health.

  • Progressive Difficulty

    20 carefully designed levels from easy to hard.

  • XP System

    Animated experience bar with level progression.
