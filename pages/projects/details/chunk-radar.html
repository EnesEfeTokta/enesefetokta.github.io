About the Project

Chunk-Radar is an English learning application designed around the chunking method - a proven technique for language acquisition that focuses on learning phrases and word combinations as meaningful units rather than individual words.

This project represents my first experience with Vibe-Coding, an AI-assisted development methodology that emphasizes rapid iteration, modern design patterns, and developer productivity.

The application combines educational psychology with modern web technologies to create an effective and enjoyable learning experience.

Key Features

  • Modern Flashcard Interface

    Single-click to play audio pronunciation (TTS), double-click to flip card and reveal translation. Clean, distraction-free study environment.

  • Visual Progress Tracking

    Color-coded mini-card navigation: Green (correct), Red (wrong), Blue (skipped), Gray (not reviewed). Click any card to jump to that chunk.

  • Analytics Dashboard

    30-day progress tracking with visual charts showing correct/wrong ratio, success rate percentage, and persistent statistics.

  • Flexible Organization

    Create custom chunk groups with modern pill-style tab navigation. Full CRUD operations with local JSON persistence.

The Chunking Method

The chunking method is a language learning technique that focuses on:

  1. 1
    Learn Phrases as Units

    Instead of individual words, learn meaningful phrase combinations

  2. 2
    Focus on Context

    Use example sentences to understand how phrases are used naturally

  3. 3
    Build Natural Fluency

    Memorize common word combinations for more natural speech

  4. 4
    Track Progress

    Use spaced repetition to reinforce learning over time

Example: "Break a leg" - A theatrical phrase meaning "good luck"
