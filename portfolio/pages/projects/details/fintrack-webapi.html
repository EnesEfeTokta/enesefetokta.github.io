Platform Overview

FinTrack isn't just an API—it's a complete SaaS backend infrastructure. Built on a modern microservices architecture, it splits responsibilities between a robust .NET 8 core for business logic and a specialized Python service for AI-powered insights.

The platform powers secure debt management, real-time budgeting, and automated financial advice via the integrated FinBot AI assistant.

Core Capabilities

  • Secure Authentication

    JWT-based auth with OTP 2-factor verification.

  • GBS Debt System

    AES-encrypted video evidence for debt verification.

  • AI FinBot

    Local LLM integration for intelligent financial advice.

  • Payment Processing

    Stripe integration for subscription and billing management.

  • Budget Analytics

    Real-time tracking with dynamic budget recommendations.

  • Multi-Account Sync

    Cross-platform data synchronization via REST API.
