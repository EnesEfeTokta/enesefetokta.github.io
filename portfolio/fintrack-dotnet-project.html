FinTrack

Windows Desktop Financial Management Application

About the Project

FinTrack is a comprehensive Windows desktop application designed for personal and small business financial management. Built with WPF (Windows Presentation Foundation) and .NET, it provides an intuitive interface for tracking expenses, managing budgets, and generating financial reports. The application serves as the Windows client for the FinTrackWebApi backend, offering seamless integration with the cloud-based financial management system.

Key Features

  • Real-time financial dashboard with interactive charts
  • Expense and income tracking with categorization
  • Budget planning and monitoring
  • Multiple account management
  • Detailed financial reports and analytics
  • Cloud synchronization with FinTrackWebApi
  • Secure local data storage with encryption
  • Export functionality (PDF, Excel, CSV)
  • Budget alerts and notifications
  • Advanced search and filtering capabilities

Technologies Used

  • .NET Framework 6.0
  • WPF (Windows Presentation Foundation)
  • C# Programming Language
  • SQLite (Local Database)
  • Entity Framework Core
  • LiveCharts2 (Data Visualization)
  • Material Design in XAML
  • AES Encryption for Data Security
  • HTTP Client for API Communication
  • iTextSharp (PDF Generation)
  • EPPlus (Excel Export)

Architecture Overview

Application Architecture

The application follows the MVVM (Model-View-ViewModel) pattern for clean separation of concerns:

  • View Layer: XAML-based user interface with responsive design
  • ViewModel Layer: Business logic and data binding
  • Model Layer: Data models and database entities
  • Service Layer: API communication and local data management

public class TransactionViewModel : ObservableObject
{
    private readonly ITransactionService _transactionService;

    public ObservableCollection<Transaction> Transactions
    {
        get => _transactions;
        set => SetProperty(ref _transactions, value);
    }
    
    public ICommand AddTransactionCommand { get; }
    public ICommand DeleteTransactionCommand { get; }
    public ICommand ExportCommand { get; }
    
    public TransactionViewModel(ITransactionService transactionService)

        _transactionService = transactionService;
