VidSync

AI-Powered Video Meeting Platform with Real-Time Collaboration

About the Project

VidSync is a cutting-edge video meeting platform that combines real-time collaboration with AI-powered assistance. Built with **ASP.NET Core** backend and **React** frontend, it provides a seamless meeting experience enhanced by the **Mistral 7B** AI model for intelligent meeting assistance, real-time transcription, and smart recommendations. The platform leverages **Redis** for high-performance caching and **PostgreSQL** for reliable data persistence, all containerized with **Docker** for easy deployment.

Key Features

  • High-quality video conferencing with real-time communication
  • AI-powered meeting assistant using Mistral 7B (via Ollama)
  • Real-time meeting notes generation
  • End-to-end encryption for secure communications
  • Redis caching for optimal performance
  • Cloudflare integration for global CDN and security
  • Responsive design for mobile and desktop

Technical Stack

Backend Technologies

  • ASP.NET Core 8.0 - RESTful API backend
  • C# - Primary programming language
  • PostgreSQL - Primary database for user data and meetings
  • Redis - High-performance caching and session management
  • Ollama + Mistral 7B - Local AI model for meeting assistance
  • JWT Authentication - Secure user authentication
  • SignalR - Real-time WebSocket communication

Frontend Technologies

  • React 18 - Modern UI framework
  • TypeScript - Type-safe JavaScript
  • WebRTC - Peer-to-peer video streaming
  • Axios - HTTP client for API calls

DevOps & Infrastructure

  • Docker & Docker Compose - Containerization
  • Cloudflare - CDN, security, and domain management
  • GitHub Actions - CI/CD pipeline
  • Nginx - Reverse proxy and load balancing

System Architecture

VidSync follows a modern microservices-inspired architecture with clear separation of concerns:

Backend Architecture

  • API Layer: RESTful endpoints for authentication, meeting management, and user operations
  • Real-Time Layer: SignalR hubs for video/audio streaming, chat, and notifications
  • AI Service: Integration with Ollama for local AI inference with Mistral 7B model
  • Data Layer: PostgreSQL for persistent storage, Redis for caching and real-time data
  • Authentication: JWT-based authentication with refresh tokens

Frontend Architecture

  • Component-Based: Reusable React components with TypeScript
  • State Management: Context API for global state management
  • Real-Time Communication: WebRTC for peer-to-peer video, SignalR for signaling

AI-Powered Features

VidSync leverages the Mistral 7B AI model through Ollama to provide intelligent meeting assistance. The AI captures conversations during meetings and automatically generates comprehensive summaries that are displayed to participants when the meeting ends.

  • Conversation Capture: AI monitors and captures meeting conversations in real-time
  • Automatic Summaries: Generates comprehensive meeting summaries shown to participants at the end
  • Key Points Extraction: Identifies and highlights important discussion points
  • Action Items: Extracts actionable tasks and decisions from conversations

Performance & Scalability

  • Redis Caching: Reduces database load and improves response times
  • WebRTC P2P: Direct peer-to-peer connections for optimal video quality
  • Cloudflare CDN: Global content delivery for fast asset loading
  • Docker Containers: Easy horizontal scaling and deployment
  • Connection Pooling: Efficient database connection management

Security Features

  • End-to-End Encryption: Secure video and audio streams
  • JWT Authentication: Stateless, secure authentication
  • HTTPS Only: All communications encrypted via SSL/TLS
  • CORS Policy: Strict cross-origin resource sharing rules
  • Cloudflare Protection: DDoS mitigation and WAF
